Base Tier
This project is a list-based app that utilizes other features that help enhance the list and help the user become more organized.

I decided to use localStorage and CRUD for making this a browser-based app, as it was simple and not too demanding.

To use this app, making a new task for the list is something that is necessary. Then, using the other features like editing it in the details button will allow the user to pull up the timer, 
just for that task, or to add a description of the task at hand. You can also sort by name, date created, or due date, and also view in either a calendar view (which shows the current week)
or by regular list view, and see what is completed and active. And if you wanted to view your timer history or wanted to make a new timer that's not connected to any task, then you can. 

I think that after personally using list and todo apps, something like this with having more features like notifications and being able to sync would make this better, but I
wanted to make this browser-based project by using localStorage and CRUD, and that meant it would only be accessible on one device at a time. Some bugs that I had during the creation of this 
project were finding small errors like broken timers, CSS bugs, and missing features, but having the AI consistently recheck itself multiple times, while it got annoying, it really helped in 
itself not hallucinating as much as it did other times.
